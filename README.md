## 📖 La Historia de ActivaChain

### 🌟 **¿Te imaginás un mundo donde las mujeres lideren la revolución digital?**

ActivaChain nació de una visión simple pero poderosa: **democratizar el acceso a la tecnología blockchain para mujeres en Latinoamérica**. No es solo otra plataforma DeFi más - es un ecosistema completo que combina educación, oportunidades económicas, y empoderamiento comunitario.

### 🎯 **El Problema que Resolvemos**

- **Barrera de entrada**: La mayoría de las mujeres no saben por dónde empezar con Web3
- **Falta de oportunidades**: Pocas plataformas están diseñadas específicamente para mujeres
- **Complejidad técnica**: La tecnología blockchain parece inaccesible
- **Falta de comunidad**: No hay espacios seguros para aprender y crecer juntas

### ✨ **Nuestra Solución**

ActivaChain es la **primera plataforma Web3 integral** que combina:

- 🎓 **Educación gamificada** desde cero
- 💰 **Marketplace descentralizado** para ganar dinero real
- 🎨 **Sistema de NFTs** para certificar logros
- 🏛️ **Gobernanza DAO** donde tu voz cuenta
- 👥 **Comunidad de mentores** verificados
- 🔐 **Reputación con ZK Proofs** para privacidad total

- **Next.js 15** - Framework React con App Router
- **TypeScript** - Tipado estático para mayor seguridad
- **Tailwind CSS** - Estilos utilitarios y animaciones
- **Framer Motion** - Animaciones fluidas y transiciones
- **Glass Morphism** - Diseño moderno y elegante

### 🌐 **Web3 Integration**
- **Wagmi v2** - Hooks de React para Ethereum
- **Viem** - Cliente TypeScript para blockchain
- **Reown (WalletConnect)** - Conexión de wallets
- **TanStack Query** - Manejo de estado y cache

### 🎮 **Gamificación & NFTs**
- **ERC721** - NFTs únicos para certificaciones
- **ERC1155** - Tokens múltiples para colecciones
- **IPFS** - Almacenamiento descentralizado
- **Sistema de Achievements** - 16 logros únicos
- **Badges de Raridad** - 8 niveles de rareza

### 🔐 **Seguridad & Privacidad**
- **ZK Proofs** - Verificación sin revelar datos
- **ReentrancyGuard** - Protección contra ataques
- **Ownable** - Control de acceso granular
- **Time Locks** - Delays de seguridad en gobernanza

## 🚀 Inicio Rápido

### 📋 **Prerrequisitos**
- Node.js 18+ 
- npm o yarn
- Git
- Wallet Web3 (MetaMask, WalletConnect, etc.)

### ⚡ **Instalación Express**

```bash
# 1. Clonar el repositorio
git clone https://github.com/M0nsxx/ActivaChain.git
cd ActivaChain

# 2. Instalar dependencias
npm install
cd frontend && npm install

# 3. Configurar variables de entorno
cp env.example .env
cp frontend/env.local.example frontend/.env.local

# 4. Desplegar contratos (opcional)
npm run deploy:sepolia
npm run deploy:arbitrum

# 5. Iniciar servidor de desarrollo
cd frontend && npm run dev
```

### 🌐 **Acceso a la Aplicación**
- **Frontend**: http://localhost:3000
- **Red Ethereum Sepolia**: Para desarrollo y testing
- **Red Arbitrum Sepolia**: Para transacciones rápidas

- **Transfer restrictions** para seguridad

#### **🎨 ActivaNFT** - Certificaciones Soulbound
- **NFTs no transferibles** (Soulbound)
- **Certificaciones de logros** y competencias
- **Integración con reputación** para bonuses
- **Metadata en IPFS** para descentralización

#### **🛒 ActivaMarketplaceMultiToken** - Marketplace Descentralizado
- **Multi-token support** (ETH, USDC, ACTIVA)
- **Sistema de reputación** para vendedores
- **Platform fees** configurables
- **Escrow automático** para seguridad

#### **🏛️ ActivaGovernance** - Sistema de Gobernanza DAO
- **Proposal system** con thresholds configurables
- **Voting periods** y execution delays
- **Quorum requirements** para decisiones importantes
- **Timelock** para seguridad
